This function generates key request and acquires license.

GenerateKeyRequest function provides following processes in conjunction with AddKey() function :

  • License Acquisition for Marlin BB
  • SAS Acquisition for MS3 After calling this function, caller should call AddKey() function for each processes.

This function supports following Marlin Protected Media Format:

  • MP4 DASH-CENC content with Marlin protection
    Marlin CDM can handle MP4 CENC media format that uses implicit mapping and explicit mapping.
Parameters
[in]session_idSession ID which is opened by OpenSession()
[in]init_dataInitialization data that is different for each initDataType.
Please refer following table to understand which initialization data will be included for each initDataType.
initDataType init_Data
"cenc" or "keyids" PSSH box
[in]initDataTypeinitialization data type information which indicates the data structure of init_data
Purpose initDataType
License acquisition with cenc initialization data "cenc"
License acquisition with cenc initialization data "keyids"
[in]keyTypeKey type which indicates process
Purpose Key Type
License Acquisition KEY_TYPE_STREAMING
[in]optionalParametersOptional information for each opration. This parameter should be always null. it is reserved for future use.
[out]requestKey request message or Device property data or ltp message.
Purpose properties in json request message
License Acquisition "httpRequest" or "completeNotification"
About detail of request message structure,
please refer marlincdm_json_request_message_eme.json.
[out]urlserver url
Return values

ERROR_NO_LICENSEValid license does not exist
