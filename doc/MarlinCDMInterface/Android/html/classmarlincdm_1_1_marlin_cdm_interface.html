This function generates key request and acquires license.

GenerateKeyRequest function provides following processes in conjunction with AddKey() function :

  • User registration for Marlin BB
  • User deregistration for Marlin BB
  • License Acquisition for Marlin BB
  • License Deletion for Marlin BB
  • SAS Acquisition for MS3
  • LTP message processing for Marlin BB After calling this function, caller should call AddKey() function for each processes.

This function supports following Marlin Protected Media Format:

  • MP4 DASH-CENC content with Marlin protection
    Marlin CDM can handle MP4 CENC media format that uses implicit mapping and explicit mapping.
  • MP4 IPMP extension content with Marlin protection
    Marlin CDM can handle MP4 IPMP content that included SINF box according to Marlin specification.
Parameters
[in]session_idSession ID which is opened by OpenSession()
[in]init_dataInitialization data that is different for each initDataType.
Please refer following table to understand which initialization data will be included for each initDataType.
initDataType init_Data
"application/vnd.marlin.drm.*" or "json" Json format data defined in marlincdm_json_initdata_android.json

In init_data, following "data_type" property is need to be specified for each purpose.
Parameter in init_data is different according to its purpose,
so caller should carefully select initDataType, keyType and init_data combination for each usage.

Purpose data_type in json init data
User Registration "atkn"
User Deregistration "atkn"
Marlin BB License Acquisition "atkn"
MS3 SAS Acquisition by SAS URL "ms3"
LTP message processing "ltp"
Get device property "prop"

please refer marlincdm_json_request_message_android.json.

This function provides metadata of Marlin BB license, SAS.
Caller should call RestoreKey() before calling this function.

Provided metadata are different by each purpose.

Purpose metadata
Marlin BB License information MBB_LICENSE_METADATA_TABLE
SAS Information MS3_SAS_METADATA_TABLE

Session ID is linked to related data by following transactions When user uses this function, provided metadata.

  • Marlin BB License acquistion transaction is executed by GenerateKeyRequest() on Android.
  • Marlin BB Registration transaction is executed by GenerateKeyRequest() on Android.
  • Marlin MS3 SAS acquisition transaction is executed by GenerateKeyRequest() on Android.
